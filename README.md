# Predicting Tetouan City Power Consumption 

Originally a project for my course Statistical Learning and Big Data. Uses regression techniques to see what factors contribute to power consumption in the city's three zones.

## Methods Used
- Multiple Linear Regression
- LASSO 
- Ridge Regression
- Random Forest
- Gradient Boosting

## Files
- tetuoancity.qmd – Quarto document with full code
- tetuoancity.hmtl – HTML file with graphs and relevant output
- dataset/ – contains the raw CSV data
